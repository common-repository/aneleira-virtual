Este plugin possibilita que o visitante do seu site descubra de maneira simples e precisa a medida de aro de anel.
É muito fácil de usar esse plugin na sua sidebar para o seu cliente descobrir o número do seu anel.

== Description ==

Descubra o tamanho do aro de seu anel de noivado, anel ou aliança com nossa Aneleira Virtual. Ela é um método preciso e prático. Você descobrirá o tamanho do aro de qualquer anel e aliança usando a tela do seu computador, basta você calibrá-la corretamente, seguindo as instruções que aparecerão na primeira etapa.
Quando a situação demanda discrição, como no pedido de casamento, por exemplo, e você quer descobrir a medida do aro dela, sem levantar suspeitas, a Aneleira Virtual é o método perfeito.

== Installation ==

= Para instalar o plugin "Aneleira Virtual" siga as instruções =

   1. Descompacte e faça o upload da pasta "aneleira-virtual", para a pasta de plugin do wordpress "wp-content/plugins/".
   2. Ative o plugin "aneleira-virtual" no painel administrativo do wordpress.
   3. Para o link do plugin aparecer na sidebar do tema, é necessário ir no painel administrativo do wordpress -> Aparência > Widgets -> Arraste "aneleira-virtual" para "Barra Lateral Principal".

   ou

   1. É necessário ir no painel administrativo do wordpress -> Plugins > Adicionar Novo -> Enviar e escolher o arquivo compactado "aneleira-virtual" em ZIP e fazer UPLOAD do mesmo.
   2. O wordpress pode pedir informações de FTP para concluir a operação, não é sempre mas acontece quando a pasta de plugin não tem permissão. Se você não tiver, peça para o administrador do blog essas informações de FTP.
   3. Ative o plugin "aneleira-virtual" no painel administrativo do wordpress.
   4. Para o link do plugin aparecer na sidebar do tema, é necessário ir no painel administrativo do wordpress -> Aparência > Widgets -> Arraste "aneleira-virtual" para "Barra Lateral Principal".
